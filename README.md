# Zero to One - A Platform Engineering Build Log

---

## What This Is

I've spent 4 years in cloud operations and change management on enterprise Splunk Cloud infrastructure - coordinating production changes, managing P1/P2 incident escalations, working daily alongside SRE and platform engineering teams. I understand how production infrastructure breaks and how it gets fixed. I just haven't been building it.

This repository documents me closing that gap - in public, in real time.

No passive learning. No collecting courses. Everything here is something I built, broke, fixed, or figured out on a terminal, IDE, TUI, Cloud UI/CLI.
